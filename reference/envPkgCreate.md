# 1st level –\> create the .pkgEnv object in Require

1st level –\> create the .pkgEnv object in Require

## Usage

``` r
envPkgCreate(parentEnv = asNamespace("Require"))
```

## Arguments

- parentEnv:

  The parent environment in which to make the new environment. Defaults
  to `asNamespace("Require")`
